What does this project do?

We set out to create a software package that takes retail store video as input and produces useful information about the foot traffic and flow as output.
For an image, we start with the left input and produce the right output.

Project Evolution

We knew what project we wanted to do from the start, and we talked about which skills each of us has to bring to the table. Feeling that we could pull this off, all four of us began researching computer vision and image processing methods for detecting people. After a while, we picked a single method and Duncan and Gabriella began to iterate on it to meet our needs. Meanwhile, Nick went out to collect sample footage from the real world, and Michael began to put together the project website you see here."
