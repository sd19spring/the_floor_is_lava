This is a live version of the code presented at https://www.pyimagesearch.com/2015/11/09/pedestrian-detection-opencv/
You must have imutils and opencv installed to run!
Use pip install imutils to install imutils (or pip install --upgrade imutils)
Use pip install opencv-python to install opencv

This tries to detect pedestrians from the current webcam feed and displays superimposed boxes in a window. Simply run ped_det_live.py to use. Chaning winStride to larger values will increase the speed.
